INSTALACION

- Ejecute Registrar.bat con privilegios elevados (click derecho -> Ejecutar como administrador).
- Explore los ejemplos segun el lenguaje.
- El driver IF Universal es compatible con puertos seriales, no asi con USB. Para conectar con impresoras USB utilice cables conversores de tipo Manhattan o en caso de las Epson utilice la herramienta Virtual Port Driver (https://download.epson-biz.com/modules/pos/index.php?page=single_soft&cid=5485&scat=36&pcat=3).

Para impresoras Epson de primera generacion se requiere instalar el paquete vb runtime.
https://sourceforge.net/projects/vb6extendedruntime/files/latest/download

EPSON TM-U220AF y EPSON TM-T900 IMPORTANTE!! 
Estos modelos de impresora deben inicializar los datos del establecimiento para poder imprimir.
Se requiere usar la herramienta de epson FPTerminal, usando los metodos SET de configuracion de establecimiento.

https://ftp.epson.com/drivers/FPTerminal.zip
